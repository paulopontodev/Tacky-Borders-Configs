# Tacky Borders Config

Configuração pessoal do **tacky-borders** para Windows.

Este repositório reúne meus arquivos de configuração do tacky-borders, usado para personalizar as bordas das janelas no Windows.

## 📁 Caminho da configuração

Os arquivos (.env | config.yaml | style.css) devem ser colocados em:

```txt
C:\Users\nome_do_seu_usuario\.config\tacky-borders
```

### Caminho alternativo para a pasta de usuário
```txt
Win + R > %USERPROFILE%
